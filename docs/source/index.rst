======================================
fall_uav: A Comprehensive UAV Library
======================================

Welcome to the fall_uav documentation. This library serves as a versatile and modular solution for UAV (Unmanned Aerial Vehicle) control, developed by the AVANT UAV team based in Brazil.

.. note:: This documentation is under active development. Contributions are welcome.

Project Overview
================

fall_uav aims to offer a "Jack of all Trades" solution for UAV control, integrating various features needed for autonomous drone operations. 

Architecture
============

The project adopts a modular architecture, designed to be both comprehensive and future-proof. It even includes an API abstraction layer for easier transition between different robotics middleware.
